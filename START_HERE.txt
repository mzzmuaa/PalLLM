Welcome to PalLLM
==================

You just unzipped a local-first AI companion for Palworld.
Everything runs on your computer; no account, no subscription.

The single thing to do
-----------------------
Double-click  play.bat

That's it. The launcher will:
  - auto-detect Palworld
  - install or refresh the PalLLM mod
  - start the sidecar
  - open the dashboard at http://localhost:5088
  - launch the game

If you'd rather see what each file does before clicking play.bat,
read PLAYER_README.txt next to this file - it has a 60-second
quickstart, prerequisites, troubleshooting, and a full file list.

Other one-click files
----------------------
  install.bat     - install the mod only (skip the launcher)
  uninstall.bat   - cleanly remove PalLLM (chat history preserved)
  support.bat     - bundle logs + health into a zip for support
  recover.bat     - last-resort reset if something is stuck

Privacy in one sentence
------------------------
PalLLM emits zero outbound traffic by default. Inference, vision,
TTS, and telemetry are all opt-in. See docs/PRIVACY.md for the
full inventory of every data-emitting surface.
